# README

This is the URB's process monitoring from PMC.

This system deals with the process management of the Secretariat of Urbanism and Works.


## Versions
* Ruby: **2.6.3**
* Rails: **6.0.0**

## Deployment
1. bundle install
2. rails db:create db:migrate db:seed
3. Done!
